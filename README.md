# Ön İşleme İş Akışı

🎯 Bu alıştırma, seni veri ön işleme iş akışı boyunca yönlendirir. Adım adım, özellik özellik veri kümesini inceleyecek ve buna göre ön işleme kararları vereceksin. Yapacakların:

- Yinelenen (duplicate) kayıtları kaldırmak
- Eksik değerlerle ilgilenmek
- Özellikleri ölçeklendirmek
- Özellikleri kodlamak (encode etmek)
- Döngüsel (cyclical) özellik mühendisliği uygulamak

Alıştırmaya başlamak için `jupyter notebook` içinde `Preprocessing-Workflow.ipynb` dosyasını aç ve yönergeleri takip et.
